# Darude Sand Cars
A simple **modern OpenGL** game made as part of Computer Graphics course work.

#### Features
 - Lighting
 - 3D Model Loading Using Assimp
 - Collision Detection based on AABB
 - Simple Physics Engine
 - Multiple Shaders Support
